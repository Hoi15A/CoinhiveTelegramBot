# Coinhivebot
A simple bot that will pull the latest pending xmr from coinhive sites as a fiat currency

## Setup

### Dependencies
- Nodejs
- git

### Install
1. Clone the repo: `git clone https://github.com/Hoi15A/CoinhiveTelegramBot.git`
2. Install dependencies: `npm install`
3. Copy and edit the config file: `cp example.config.json config.json && vim config.json`
4. Run the bot: `npm start`
